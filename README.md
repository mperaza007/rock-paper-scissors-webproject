A simple Spring Boot project that hosts a rock-paper-scissors game on a local server on port 8080. The game is a simple rock-paper-scissors game where the user choose from the 3 options and the computer takes a random uption.
Technologies: Java, Spring Boot, HTML, CSS, JavaScript.
